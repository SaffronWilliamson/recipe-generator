**Project name:** Recipe Generator

**Description:** Recipe Generator is a Java console-based programme that creates personalised recipes for users based on their dietary preferences (vegetarian or not), spice tolerance, and calorie requirements. The application guides the user through a series of questions, generates a matching recipe, and optionally provides a random bonus recipe. 

**Features:**
- User-friendly console interface
- Personalised recipe generation based on user preferences
- Option to generate a random bonus recipe
- Easy-to-use and intuitive design
- Customisable and expandable for future enhancements
- Supports multiple dietary requirements and spice levels
- Object-oriented design with inheritance for Meat and Vegetarian recipes

**File Structure:**
- RecipeGeneratorApp.java: Main application class
- Recipe.java: Abstract class representing a recipe
- GreetingPage.java: Class for displaying the greeting page
- UserInputHandler.java: Class for handling user input
- MeatRecipe.java: Concrete class representing a meat recipe
- VegetarianRecipe.java: Concrete class representing a vegetarian recipe
